# Deployment Checklist (Ops Guide)

## Pre-Deployment
- [ ] 1323 director approval (verbal)
- [ ] IT team access ready
- [ ] PDPA consent drafted
- [ ] Escalation SLAs defined
- [ ] Training materials ready
- [ ] Backup plan documented
- [ ] Go/no-go approved

## Deployment Day
- [ ] System health check (uptime, response time)
- [ ] Load test (can handle 50 concurrent?)
- [ ] Database connectivity verified
- [ ] Logging enabled (audit trail)
- [ ] Escalation webhook tested
- [ ] Demo scenario working
- [ ] Team briefed

## Training (2 Hours)
- Part 1: What is NamoNexus? (30 min)
- Part 2: Workflow demo (30 min)
- Part 3: Safety and escalation (30 min)
- Part 4: Q&A and practice (30 min)

## Go-Live
- [ ] Start with 10 calls/day (test)
- [ ] Monitor every interaction
- [ ] Have team on standby
- [ ] Document any issues
- [ ] Daily standup first week

## Success Criteria
- [ ] Zero critical errors in first 50 calls
- [ ] Counselors comfortable with workflow
- [ ] System uptime 99%+
- [ ] Response time < 3 sec
- [ ] Escalation working 100%
