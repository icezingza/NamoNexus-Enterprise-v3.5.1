# Identity Capsule Directory

This directory stores identity files for NamoNexus Enterprise.

## Purpose

The identity capsule contains sensitive configuration and identity information
for the system. This directory should be properly secured and backed up.

## Contents

Files in this directory may include:
- Identity certificates
- Encryption keys  
- Authentication tokens
- System identification data

## Security

⚠️ **IMPORTANT**: This directory contains sensitive information.
- Limit access to authorized personnel only
- Ensure proper file permissions are set
- Regularly backup this directory securely
